Installation
============

This package is still in its development phase, but you can compile the package from source

.. code-block:: bash

  git clone https://github.com/factoryofthesun/mlisne
  cd mlisne
  pip install .

To install in development mode

.. code-block:: bash

  git clone https://github.com/factoryofthesun/mlisne
  cd mlisne
  pip install -e ./
